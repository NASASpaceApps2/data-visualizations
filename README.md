## Backend API

### Run locally
1. Clone this repository
2. Create .env and add `MONGO_URI=<MongoDB URI>`
3. Run `npm run i` and `npm run start`

### Deploying to heroku
1. Clone this repository
2. Add `MONGO_URI` as an environment variable
3. `npm run deploy`
